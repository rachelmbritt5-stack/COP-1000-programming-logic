## Purpose
My project is a based on the American pastime, baseball. The project is comprised of two quizzes. The first quiz tests the user’s ability to name teams based on their patron city. The second quiz tests the user’s ability to identify certain baseball players based on a clue. The program prompts the user to select which quiz he would like to play. In either quiz, when the user types in the correct answer he is awarded a hit, which means he has answered the question correctly, or a strike, when he incorrectly answers the question. The user is limited to three strikes. When he executes his third strike, the user is out, his score is displayed, and the quiz ends. Each quiz has a limited number of questions; therefore, when the limit is reached, the user’s score is displayed, and the quiz ends. Either way, once the quiz ends, the program will prompt the user to make a quiz selection. The purpose for the project is to test the user’s knowledge of baseball in an entertaining way.
