## Purpose
The project I have made is a baseball quiz. This will test the user’s ability to name teams based on its locations and certain players based on a clue. The user will be prompted to select which quiz they would like to play first. While going through the quiz the user will be awarded a hit, which means they answered the question correctly, or a strike, which means they answered the question incorrectly. The user is limited to only three strikes, if they reach three strikes, they are out, and the quiz ends. The quiz will also end if it runs out of questions. At the end of the quiz, it will display the user’s final score and take them back to the main prompt where they can play the quiz or play the other one. This project is pure for enjoyment and entertainment.
